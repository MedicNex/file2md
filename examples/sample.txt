# MedicNex File2Markdown 示例文档

这是一个示例文档，用于演示 File2Markdown 服务的功能。

## 功能特性

- 支持多种文档格式转换
- 智能图片识别和OCR
- API Key 安全认证
- 高性能异步处理

## 使用方法

1. 上传文档文件
2. 获取 Markdown 格式输出
3. 集成到您的应用中

### 支持的格式

| 类型 | 扩展名 | 说明 |
|------|--------|------|
| 文本 | .txt, .md | 纯文本和Markdown |
| 文档 | .docx, .doc | Microsoft Word |
| 表格 | .xlsx, .xls, .csv | Excel和CSV |
| 演示 | .pptx, .ppt | PowerPoint |
| PDF | .pdf | PDF文档 |
| 图片 | .png, .jpg等 | 各种图片格式 |
